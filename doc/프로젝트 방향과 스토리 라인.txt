1. 커버페이지 서울 지도에 일러스트를 씌워서 (like 소곳소곳)궁 표시 
   or 궁궐 위치 일러스트 사용해서 각 궁궐 표시
-> 각 궁궐 이미지 버튼 클릭시 각 궁 소개 페이지로 들어갈 수 있게
-> 메인 페이지 버튼 메인페이지 이동

2. 메인페이지 
-> 헤더 : 로그인 => 사용자-> 소셜로그인    관리자
-> 탑   : 궁능소개 / 통합예약 / 행사 / 공지사항 / 소개(팀 프로젝트 개요 등)
				: 호버시 쭉 리스트 보여주기
				: 궁능소개 등 탑 카테고리 클릭시에는 이동 없고, 궁 이름 클릭시 해당 궁 페이지로 이동
-> 메인 :  스프레드
-> 풋  : 통합공지사항 / 행사포스터 슬라이드 (일시정지,재생버튼)
-> 왼쪽 탭 : 전체 궁궐 동그라미표시버튼(클릭시 각 궁궐 소개, 역사, 오시는길, 예약) + 깃 레포지토리
-> 해더,탑,풋,왼쪽 탭은 고정

3. 각 궁궐 상세안내 페이지
->  메인 궁궐 이미지
		**궁 소개/ 역사 
		**궁 이야기
		**궁 관람안내 
		통합예약

		https://royal.cha.go.kr/ROYAL/contents/R101000000.do	  
		이미지 아래 관람도우미를 관람코스 위치로 같은 형식으로 진행
		관광 코스 -> 관람 안내로 변경 
		
3-1> 궁 소개/ 역사
3-2> 궁 이야기 (문화재 API 사용)
3-3> 궁 관람안내 (관람안내 클릭시 해당 궁 내용만 뿌려주기(관람시간, 관광코스, 관람요금, 관람규칙, 오시는길))
3-4> 통합 예약 페이지
