   - 슬라이드와 공지사항 간 간격 수정
   - 행사 이미지 클릭 시 새창으로 연결되게 수정

(조현우)
>  해설예약 page
   - 관람안내page에 있는 정기휴일날짜에는 예약을 할 수 없게 막아 놓음 
   - 해설예약 페이지에서 예약 완료후 mypage로 보냄
>  궁소개 page
   - 각각의 궁 인덱스 페이지 해설 행사 예약을 해설 예약으로 수정
>  궁 내부 건축물 page
   - 상단에 궁 선택하는 부분 다시 작업
   - 타이틀을 xxx 이야기에서 xxx 내부 건축물로 변경
   - 궁 내부 건축물 메인 이미지가 창 크기 변화에 따라 비율이 어긋나는 부분 수정
   - 들어가는 텍스트들의 전체적인 디자인 수정
> 행사정보 page
   - 행사정보 페이지에 행사 링크로 연결시 새창으로 연결
> 관람안내 page
    - 메인타이틀 (오시는길) 추가
    - (xxx 오시는 길) 서브 타이틀 추가
    - 궁 네비게이션 수정
    - 전체적인 디자인 수정
    - 전체적인 수정을 하면서 js / css 코드들 정리


(김준형)

   - 페이지의 크기가 작아질때 royal navibar가 페이지내의 navicontainer를 침범하지 않게 수정

(김태형)
> 마이페이지 page
   - 발송된 이메일 <<예약 상세 내용>> 이 <> 로 나오던 것 수정
   - 예약내역에 날짜및시간 데이터 수정
   - 예약 취소 시 날짜/시간이 지난경우 예약취소 불가 처리
   - 예약 취소 버튼 디자인 변경
   - 예약 취소 시 취소확인 alert 창 확인 후 예약취소 되게 수정
